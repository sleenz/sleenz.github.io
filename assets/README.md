# Assets Folder

This folder contains all the media files and documents for your portfolio website.

## Required Files

Please add the following files to this folder:

### Images

1. **profile.jpg** - Your profile/headshot photo (recommended size: 400x400px)
2. **project1.jpg** - Screenshot or image for Project 1 (recommended size: 600x400px)
3. **project2.jpg** - Screenshot or image for Project 2 (recommended size: 600x400px)
4. **project3.jpg** - Screenshot or image for Project 3 (recommended size: 600x400px)

### Documents

1. **cv.pdf** - Your curriculum vitae/resume in PDF format

## Notes

- If you don't add these files, placeholder images will be shown automatically
- You can add more project images by naming them appropriately (project4.jpg, project5.jpg, etc.)
- Supported image formats: .jpg, .jpeg, .png, .webp
- Keep image file sizes reasonable (under 1MB each) for faster loading

## Adding Your Files

1. Download/prepare your images and CV
2. Place them in this `assets` folder
3. Make sure the filenames match exactly (case-sensitive)
4. Commit and push the changes to GitHub
